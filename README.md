# Katy Baron Studio - Permanent Makeup Website

A premium, multilingual (EN/RU/UK) website for a luxury permanent makeup studio.

## Features
- **Modern Design**: Premium aesthetics, smooth animations, and responsive layout.
- **Multilingual Support**: Integration with i18n for English, Russian, and Ukrainian.
- **Dynamic Services**: Detailed descriptions of specialized procedures.
- **Interactive Gallery**: Portoflio with filtering and lightbox.
- **Pricing**: Clear, categorized service menu.
- **Booking & Contact**: Easy-to-use forms for client engagement.

## Tech Stack
- **HTML/CSS/JS**
- **Vite** (Build tool)
- **TypeScript**
- **i18next** (Translations)

## Local Development
1. Install dependencies: `npm install`
2. Run development server: `npm run dev`
3. Build for production: `npm run build`

## Deployment
The site is optimized for deployment on hosts like Hostinger. See `hostinger_deploy_guide.md` for details.
